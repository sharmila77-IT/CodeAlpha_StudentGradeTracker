# CodeAlpha_StudentGradeTracker
Java Student Grade Tracker using OOP and ArrayList

## Project Description
This is a Java-based application to manage student grades.

## Features
- Add student details (name & marks)
- Calculate average marks
- Find highest and lowest marks
- Grade calculation (A–F)
- Menu-driven system

## Technologies Used
- Java
- OOP Concepts
- ArrayList

## How to Run
1. Compile the program:
   javac StudentGradeTracker.java

2. Run the program:
   java StudentGradeTracker

## Output
Displays student details and performance report.
